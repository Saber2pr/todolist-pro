### Introduction

The Search feature lets you quickly find tasks across all your `.todo` files in a directory. If you have many files, this is the fastest way to locate a specific task.

> Find any task instantly!

### How to Use

1. Open the **Search** page from the toolbar or navigation menu.

2. Type a keyword in the search bar and press **Enter**.

3. The app will scan all `.todo` files in your current directory (up to 10 levels deep) and show matching results.

4. Each result shows the file path and matched content. Click on a result to open that file directly.

### Tips

- Search works across **all** `.todo` files in your workspace, not just the currently open file.
- Use specific keywords for more accurate results — e.g., search for "meeting" instead of "work".
- The result count is displayed at the top so you know how many matches were found.
