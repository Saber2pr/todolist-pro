### Introduction

Todolist automatically saves a version of your file every time you make an edit. You can browse your full edit history, compare changes, and restore any previous version with one click.

> Never lose your work — every edit is saved!

### How to Use

1. Open your todo file, then go to **Settings > Backup** to see all saved versions.

2. Each version is listed with a timestamp (date and time), so you can find exactly when a change was made.

3. Click on any version to preview it.

### Comparing Versions

When you select a version, the app shows a **side-by-side diff view**:

- **Left side**: your current file
- **Right side**: the selected backup version
- Lines highlighted in **green** are additions, **red** are removals

This makes it easy to see exactly what changed.

### Restoring a Version

1. Find the version you want to go back to.
2. Click the **Undo** (↩) icon next to that version.
3. Confirm the restore — your file will be reverted to that version.

### Managing Backups

- The app shows the total backup size (in MB) at the top.
- If backups grow too large, you can use **Clear Backup** to free up space.
- Backups are stored locally on your computer for fast access.

### Tips

- Backups are created automatically — no need to manually save.
- Use the diff view to quickly find when a task was added or removed.
- This feature is available for **Pro** users.
