### Introduction

Sharing lets you generate a link to your todo list that others can open and view — without needing an account or the app installed. The shared view is **read-only**, so your tasks are safe from edits.

> Share your plans with anyone via a link!

### How to Use

1. Open the todo file you want to share.

2. Click the **Share** button in the toolbar to generate a share link.

3. Copy the link and send it to anyone — via email, chat, or wherever you like.

4. When they open the link, they'll see your todo list in a clean, read-only view.

### What the Viewer Sees

- The full todo tree structure with all tasks and sub-tasks
- Task status (done / not done)
- No editing, settings, or backup controls — view only

### Tips

- Great for sharing project plans, meeting agendas, or study lists with teammates.
- The viewer does not need to install anything — the link opens in any web browser.
- If you update your tasks after sharing, the viewer will see the latest version when they refresh.
